# RoboMaster SDK Documentation

Collection of guides and examples for DJI RoboMaster development.

## Project Structure

```
RoboMaster-SDK-Docs/
├── AI/                     # Computer Vision & Object Detection
├── setup/                  # Installation & Configuration
└── README.md
```

## Quick Start

1. Follow [Setup Guide](setup/) to install RoboMaster SDK
2. Test connection: `python setup/check_connection.py`
3. Explore AI examples in [AI directory](AI/)

## Categories

### [Setup](setup/)
- Python 3.8 & RoboMaster SDK installation
- Virtual environment setup
- Connection testing

### [AI](AI/)
- YOLOv8 object detection
- Automated data collection
- Model training & inference
